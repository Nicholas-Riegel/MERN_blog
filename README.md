# MERN-Bootstrap Blog

A MERN stack blog displaying full CRUD operations with Bootstrap styling.

Dependencies: 
* React 
* React-router-dom 
* Axios 
* React-bootstrap 
* Concurrently
* cors 
* dotenv 
* express 
* mongoose

To run: 

1. Download project files
2. npm install
3. cd into frontend
4. npm start
